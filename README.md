This repository contains code samples for working with the Cal-Adapt API.


## Setup
- Clone the repo

	```python
	git clone https://github.com/berkeley-gif/caladapt-docs.git
	```
- Change directory

	```python
	cd caladapt-docs
	```
- Create a python3 virtualenv

	```python
	python3 -m venv env
	```
- Install libraries

	```python
	pip install -r requirements.txt
	```
